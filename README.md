# 360 For Business Website

A professional website for 360 For Business - a full-service marketing and technology company offering an all-in-one CRM platform and complete marketing solutions.

## Features

- **Homepage** with interactive toggle between Complete CRM and Complete Marketing sections
- **CRM Platform** page showcasing 6 core features
- **Pricing** page with Starter ($79/mo) and Pro ($149/mo) plans
- **Marketing Services** page with 6 service categories
- **Contact** page with form integration
- **Cost Savings Calculator**
- **Feature Comparison Matrix**

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS 4
- tRPC for type-safe APIs
- Drizzle ORM for database
- Vite for development

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```
